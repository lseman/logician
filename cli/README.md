# logician-ink-cli

A TypeScript + React + Ink terminal app inspired by Claude Code UX.

## Features

- Chat transcript area
- Fixed input + status footer area style
- Status phases: `thinking`, `bubbling`, `jambering`, `streaming`, `ready`
- Live trace panel toggle with `Ctrl+T`
- Slash commands (`/help`, `/status`, `/doctor`, `/bug`, `/clear`, `/trace`, `/quit`)

## Run

```bash
cd cli
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
